## 增加命令
### 增加模块
```bash
$ nest g m xxx
$ nest g m flow/xxx
```
### 增加控制器
```bash
$ nest g co xxx
$ nest g co flow/xxx
```
class（别名：cl）
控制器（别名：co）
装饰者（别名：d）
例外（别名：e）
过滤器（别名：f）

警卫（别名：顾）
拦截器（别名：i）

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
